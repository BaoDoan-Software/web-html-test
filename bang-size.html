Cách đo size chân

Để chọn size vớ chính xác, bạn cần biết size giày của mình. Size vớ NOIR tương ứng với size giày như bảng bên dưới.

  • Đặt chân lên tờ giấy, vẽ viền bàn chân và đo chiều dài từ gót đến ngón dài nhất
  • Hoặc dùng trực tiếp size giày bạn đang mang hằng ngày

Lưu ý khi chọn size

  • Nếu size giày của bạn nằm giữa hai nhóm, nên chọn size lớn hơn để thoải mái
  • Vớ thể thao nên chọn vừa khít để tránh trượt khi vận động
  • Vớ len mùa đông có thể hơi co lại sau khi giặt, nên chọn đúng size hoặc tăng 1 size
  • Set vớ 7 ngày có chung một size, phù hợp mua làm quà tặng
